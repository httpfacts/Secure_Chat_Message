Secure Chat Messenger

A demo that shows message-level encryption using AES-GCM (password → PBKDF2 → AES). Each message uses a fresh salt & IV so every ciphertext is unique.

Open Secure Chat Demo →

Why this project?

  • Applies AES practically — each message is encrypted with the sender’s password-derived key.
  • Receiver can decrypt only if they know the correct password (salt + IV are embedded).
  • Clear visual of locked/unlocked messages — great for demonstrating confidentiality in class.

Tip: Use different passwords to show failed decrypts, or the same password to show successful decrypts.
